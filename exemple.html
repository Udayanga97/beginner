Notations short-hand

  • font = style weight size line-height family
  • background = color image repeat attachement position

Sélecteurs CSS

  • * : pour tout sélectionner
  • , : pour ajouter un autre élément
  • plusieurs éléments : pour cibler le dernier
  • + : pour cibler aussi l'élément suivant de meme niveau
  • > : permet de sélectionner tous les éléments childs désignés d'un élément
  • [] : permet de séléctionner les attributs avec leur valeurs ex: [class="para"]

Speudo classes

  • :link : lien non visité
  • :visited : lien visité
  • :hover : au survol de la souris
  • :nth-child : premier enfant d'un élément
  • :last-child : dernier enfant d'un élément
  • :first-of-type : premier type d'élément
  • :last-of-type : dernier type d'élément
  • :nth-child() : énième enfant d'un élément (valeurs possible: odd:impair even:pair ou chiffre nombre

  • ::before : avant
  • ::after : après

Formulaires

Types d'éléménts

  • input : sert a entrer une information
  • label : description de l'information
  • textarea : texte sur plusieurs lignes
  • optgroup : permet catégoriser une liste
  • option : point de liste
  • fieldset : sert à catégoriser l'élément
  • legend : sert à nommer la catégorie de l'élément

Type d'inputs

  • text : champs de caractère sur une seule ligne
  • radio : pour séléctionner un seul choix
  • checkbox : pour séléctionner plusieurs choix
  • number : pour entrer un nombre
  • email : pour entrer une adresse mail
  • url : pour entrer une adrese internet
  • date : pour entrer une date
  • password : Pour entrer un mot de passe
  • tel : numéro de téléphone
  • image : choisir une image (src)
  • reset : reset du formulaire par défault

Types d'attributs

  • maxlength : nombre maximum de caractère autoriser
  • minlength : nombre minimum de caractère autoriser
  • name : pour faire le lien avec php
  • for(label) / id(input) : sert à mettre en lien le label et l'input
  • autocomplete : remplissage du champs automatique ou non / on off
  • autofocus : focus sur le champ a l'affichage du formulaire
  • required : champs obligatoire à l'envoi du formulaire
  • checked : présélection par défault(checkbox)
  • selected : présélection par défault(list)
  • placeholder : indication d'information
